## What is this project?

This is a desktop application developed as a 2nd-year university project for the Human-Computer Interaction (IPC) course. It is a personal finance management tool designed to help users track their daily purchases, organize expenses by categories, and visualize their spending habits.

> **Note:** This is a legacy academic project focused on learning UI/UX principles, event-driven programming, and desktop application architecture using Java.

## Tech Stack

*   **Programming Language:** Java.
*   **UI Framework:** JavaFX.
*   **Interface Design:** JavaFX Scene Builder (FXML).
*   **Build Tool:** Ant.

## Running the Project

If you want to explore the source code or play the build locally:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/davidvilaa/proyecto_ipc.git](https://github.com/davidvilaa/proyecto_ipc.git)
   ```
2. **Open in your IDE:**
   Import the project into an IDE with JavaFX and Ant support (such as NetBeans, IntelliJ IDEA, or Eclipse).
3. **Run:**
   Execute the main file to launch the user interface and interact with the application.
